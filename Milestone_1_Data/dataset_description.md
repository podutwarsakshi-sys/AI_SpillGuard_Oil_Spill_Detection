# Dataset Description

The dataset used in this project consists of satellite SAR images
for oil spill detection. Each image has a corresponding segmentation
mask indicating oil spill and non-oil regions.

The dataset is divided into:
- Training set
- Validation set
- Test set

The dataset is used for training and evaluating a deep learning
segmentation model.
